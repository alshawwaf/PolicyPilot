How "Try it out" & Export work

Try it out runs through the portal by default (server-side), so it reaches your saved servers without the browser's CORS block — just Authorize with a session id (sid, auto-captured after you run login) and execute. The proxy only reaches servers you've added, and TLS is verified server-side. Untick “Run calls through the portal” to call straight from the browser (handy for the copy-paste curl; a real server then shows “Failed to fetch — CORS”, which is expected). For governed changes, use the gated editor on Management Servers (preview → publish).

Export for Postman / Bruno downloads this version's OpenAPI 3.0 file with your target baked in. Postman: Import → File. Bruno: Import Collection → OpenAPI V3. Re-export after switching the version or target.
