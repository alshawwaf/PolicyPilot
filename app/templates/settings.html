Changes take effect immediately, across the whole portal.

API keys

Named, revocable keys for the machine endpoints. An MCP key is sent as Authorization: Bearer <key> to /mcp; a webhook key is sent as the X-DCSim-Token header. Keys are stored hashed and shown once at creation — copy it then. Revoking is immediate. (The single shared token in MCP / agent and Ticketing webhook above still works too, as a fallback.)
